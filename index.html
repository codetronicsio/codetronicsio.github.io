Some of our works // Algunos Trabajos Realizados

  • Development of a web application for the storage and processing of lottery results. // Desarrollo de una aplicación web para el almacenamiento y procesamiento de los resultados de la lotería.

  • Contract with a US startup to be its main development team. The first version of a mobile application for IOS and Android was implemented. It allows the treatment monitoring of chronic diseases. // Contrato con una startup de EEUU para ser su principal equipo de desarrollo. Se implementó la primer versión de una aplicación móvil para IOS y Android la cual permite realizar el seguimiento de tratamientos de enfermedades crónicas de pacientes.

  • UI upgrade project for a US based robotics company. The improvement of an existing robot-human interface was done, adding new functionalities and look & feel. // Actualización de la interfaz de usuario para productos de una empresa de robótica de EEUU.

  • Collaborative working with a Canadian robotics company in a robot control interface. // Trabajo conjunto con una empresa de robótica Canadiense en una interfaz de control de robots.

  • Recovery of undocumented products for an Argentine company. Their hybrid applications of commercial promotions were refunctionalized for a presentation to new investors. // Recuperación de productos indocumentados para empresa argentina. Se refuncionalizaron aplicaciones híbridas de promociones comerciales para ser presentadas a inversionistas.

  • Analysis, design, development and testing of a tool for Linux and Windows for a local software company. Through it, the recording and real time streaming of audio and video of a device can be performed. Control and health status interfaces were provided. This media tool will be used by a mobile phone provider in South America. // Análisis, diseño, desarrollo y testeo de una herramienta para Linux y Windows para empresa local de software. Mediante ella, se puede realizar la grabación y transmisión en tiempo real de audio y video de un dispositivo. Provee interfaces de control y de chequeo de estado. Será usada por proveedor de telefonía móvil en Sudamérica.

  • Development of the user interface of a local medical software products company. A hybrid application for IOS and Android was built. It allows the monitoring and control of medical treatments. It has a chat for effective communication between patients and health professionals as well as different interfaces to obtain data from third-party medical equipments. // Desarrollo de la interfaz de usuario de productos de empresa local de software médico. Se construyó una aplicación híbrida para IOS y Android la cual permite el seguimiento y control de tratamientos médicos. Dispone de un chat para la efectiva comunicación entre pacientes y profesionales de la salud como así también distintas interfaces para obtener datos de equipos médicos de terceros.

  • Technology upgrade project for a US based robotics company. Migration of the existing robot-human mobile and web interface to Facebook React. // Actualización de tecnologías usadas en proyectos de robótica de EEUU. Se migró la interfaz de usuario web y móvil existente a Facebook React.
